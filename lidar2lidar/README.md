# lidar2lidar

This module implements the raw-record multi-LiDAR calibration workflow used in
this repository.

## Commands

- `lidar2lidar-topics`
- `lidar2lidar-auto`
- `lidar2lidar-calibrate`
- `lidar2lidar-extract`
- `lidar2lidar-merge`
